### 입출력 코드 예제

```
#include <iostream>

using namespace std;

int main()

{
    int age;
    cout << "여러분의 나이를 입력해 주세요 : ";
    cin >> age;

    cout << "여러분의 나이는 " << age << "살 입니다." << endl;
    return 0;

}
```

#### iostream : 입출력 클래스, cout/cin 사용하려면 해야함

#### cout {'<<'} var_output

#### cin {'>>'} var_input
