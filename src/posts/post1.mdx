### 입출력 코드 예제

```
#include <iostream>

using namespace std;

int main()

{
    int age;
    cout << "여러분의 나이를 입력해 주세요 : ";
    cin >> age;

    cout << "여러분의 나이는 " << age << "살 입니다." << endl;
    return 0;

}
```
